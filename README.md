 # 🚀 HERO.IO - Modern App Store Platform

**HERO.IO** is a high-performance, responsive app store discovery platform built with **React.js** and **Tailwind CSS**. It provides a seamless user experience for browsing diverse application categories, viewing technical details, and managing an installation queue.

## ✨ Key Features

- **Dynamic App Ecosystem:** Browse through 14+ curated app categories with real-time data.
- **Detailed Insights:** Dedicated dynamic routes (`/apps/:id`) for every app, featuring comprehensive descriptions, size, and download stats.
- **Installation Queue:** A robust system using `LocalStorage` to add, track, and remove apps from your personal queue.
- **Interactive Ratings:** Visual rating breakdowns using professional progress bars and average rating displays.
- **Responsive UI:** Fully optimized for Mobile, Tablet, and Desktop using a "Mobile-First" approach.
- **Premium Dark Mode:** Sleek aesthetic featuring glassmorphism effects and high-contrast typography.


src/

├── hompage/         # Specific sections for the landing page
├── assets/          # Static images and global styles
└── App.jsx          # Routing configuration and global providers

## 🛠 Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS, DaisyUI
- **Icons:** React Icons
- **Notifications:** React-Toastify
- **State Management:** React Hooks (useState, useEffect)
- **Routing:** React Router v7
- **Data Persistence:** Browser LocalStorage
